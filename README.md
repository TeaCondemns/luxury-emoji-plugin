[[Download plugin]](https://github.com/TeaCondemns/luxury-emoji-plugin/raw/main/luxury-emoji-plugin-1.18.2.jar)

[[Luxury Emoji resource pack GitHub]](https://github.com/TeaCondemns/luxury-emoji)
